## cron

`/var/spool/cron/crontab/USER` : **USER**'s **crontable**  
`/etc/crontab` : sys tasks  
*	usually calls all `cron.{hourly|daily|weekly|monthly}`

`/etc/cron.{hourly|daily|weekly|monthly}` : sys tasks, divided by common planning periods
