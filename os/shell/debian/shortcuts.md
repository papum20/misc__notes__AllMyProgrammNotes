# SHORTCUTS

`Ctrl+Alt+F<x>` : open virtual terminal number `x`  

## Signals
Terminal interprets the following as signals.  

`Ctrl+Z` : `SIGTSTP`  
`Ctrl+C` : `SIGINT`  
`Ctrl+\` : `SIGQUIT`  

## Stream control
`Ctrl+D` : `eof`  
`Ctrl+Q` : `start`  
`Ctrl+S` : `stop`  

## Terminal
`Ctrl+R` : search command history
